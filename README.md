# Nightbot API

A custom API built for Nightbot and YouTube livestreams.

## Features

- 🎭 Random Shayari
- ❤️ Love Quotes
- 😂 Jokes
- 😍 Pickup Lines
- 🧠 Riddles
- 💡 Quotes
- 🎮 BGMI Commands
- 🤖 Interactive Chat Commands
- 📜 Random Facts
- 🌙 Good Morning / Good Night Messages

## Tech Stack

- Cloudflare Workers
- GitHub
- JSON Database
- JavaScript
